YOLO作为最流行的目标检测算法，ultralytics（[https://github.com/ultralytics/ultralytics]）引领YOLO的潮流，但是作为一个功能强大的库，ultralytics的代码具有较大的复杂性，
不便于研究人员在此基础上进行修改。因此我将ultralytics的源码进行修改，仅保留YOLO11目标检测的部分，即YOLO-Lite，并经过测试保证代码可以正常运行。
